# Magic RDR v1.4.0

RPF Editor for Red Dead Redemption (no longer in active development)

This tool supports the Xbox 360, Nintendo Switch, PS4 & PC platforms.                   
Some features are only supported for the Xbox version which was the main target originally.

# Features
 - Add/remove files/directories
 - Basic Hex viewer
 - Basic Hash generator
 - Basic Sector data viewer (#SI)
 - Basic Shader data viewer (.FXC, .NVN)
 - Script decompiler (#SC)
 - Texture viewer & editor (#TD, #SF, #FT, #FD, #VD)
 - Model viewer (#FT, #FD, #VD, #BD)
 - Stringtable viewer (#ST)
 - Audio player (.AWC)
 - **Encrypt TOC** option: Control whether the Table of Contents is encrypted when saving RPF files
 - **Encrypt Data** option: Control whether resource data is encrypted/compressed when saving RPF files (allows saving files in decrypted/decompressed state)
 - **Dark Mode**: Modernized dark theme user interface
 - **Subfolder Support**: Enhanced handling of nested directories and recursive folder operations

# Usage

### Creating Subfolders
To create a new subfolder within an RPF archive:
1. **Right-click** on the directory list or file view where you want the new folder.
2. Select **Create Directory** from the context menu.
3. Enter the desired name for your new folder.
   - **Tip:** You can create multiple nested folders at once by entering a path like `audio/sfx/cutscene`. The tool will automatically create the entire directory structure for you!
4. You can also **Import Directory** to recursively add an entire folder structure from your computer.

### Enabling Dark Mode
1. Go to **Options** > **Settings** in the main menu.
2. Check the **Dark Mode** box.
3. The interface will instantly switch to the dark theme.
